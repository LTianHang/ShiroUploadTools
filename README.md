## 研发此工具起因

某次项目中 遇到一个Shiro环境 无法打入内存马 低权限账户 无法写入文件(echo `各种马的危险函数会被拦截` 无法落地) 

后修改Shiro利用代码为文件落地生成类 后续成功写入

已整理为工具
